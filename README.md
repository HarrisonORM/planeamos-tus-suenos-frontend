# ✦ Planeamos tus Sueños — Frontend

Sitio web de empresa de organización de eventos ubicada en el Oriente Antioqueño, Colombia (Rionegro, Guarne, Llanogrande).

## Sobre el proyecto

Aplicación web completa para una empresa de planeación de eventos que incluye:

- Portafolio de eventos realizados con galería filtrable
- Catálogo de alquiler de mobiliario, menaje, sonido e iluminación
- Contratación de servicios (meseros, chef, DJ, repostería, fotografía)
- Paquetes prediseñados con cotizador interactivo
- Recomendación de locaciones en el Oriente Antioqueño
- Sistema de reservas con anticipo a través de pasarela de pagos
- Formularios de contacto y cotización de eventos

## Stack tecnológico

- **Framework:** Next.js 16 con App Router
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS 4
- **Imágenes:** Cloudinary
- **Pagos:** Wompi (sandbox)
- **Despliegue:** Vercel

## Instalación

```bash
git clone https://github.com/HarrisonORM/planeamos-tus-suenos-frontend.git
cd planeamos-tus-suenos-frontend
npm install
```

Crea un archivo `.env.local` basándote en `.env.local.example` y completa las variables.

```bash
npm run dev
```

Abre http://localhost:3000
